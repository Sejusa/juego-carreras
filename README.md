ES: Juego de carreras de recorrer 50 casillas, el primero en llegar gana. Incluye 3 tipos de objetos y bonus de avanze/retroceso. No permite jugar más allá de coger si utilizar un objeto o no, es un proyecto universitario que muestra los conocimientos hasta ahora.

EN: A race game where you run 50 spaces; the first to arrive wins. Includes 3 types of objects and a forward/backward bonus. The gameplay is limited to choosing whether or not to use an object; it's a university project showcasing current knowledge.
